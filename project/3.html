Project Links

Michigan Legal Help website is created to help people who have to handel simple civil legal problems without a lawyer. The website is an interactive, statewide website that provides legal information and referrals to self-represented litigants. The site is built on drupal becasue drupal provides well-functional data structure and models to manage content.

Responsive Design

The sites has a lot of toolkits which includes complex condition filters and automated forms. Because of this, the drupal needs a lot of views and blocks with conditional statements. In the development, all content is managed by several different containers, such as blocks, modules, views, context and content blocks.
