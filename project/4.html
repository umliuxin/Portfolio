Job Role: Web Development
Time: June 2014 - Oct 2014
Main Roles: Webpage Design, Responsive Design, Web Development
Main Tools: Drupal, Illustrator, Photoshop, HTML/Css/JS

My Main role is to build a new version of webpage of United States Geographic Survey Great Lake Science Center. The original webpage is traditional informatic webpage to present station's news, publications and events. My target is to implement lastest web development technology and design ideas to build a more user friendly website which makes the resource more valuable and easier to be accessed.

Nuwa Prototype

During Designing, My teammates and I focused on how to keep our content attractive to public. In order to understand what content is most useful and valuable to users, we did a comparative analysis to compare our strength and weakness between other similar websites. Also we considered different user group and how they access to the content like tools or reports.

During Interview

In the development process, my main task is to keep all pages responsive on different width of screens. In order to implement this, I used drupal themes to set up css media query. Meanwhile, all content is wraped and controlled by blocks, views, modules and pages. Some images and buttons were optimized for different user experience, such as touch screen.
